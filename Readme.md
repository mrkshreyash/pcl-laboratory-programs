# PCL Laboratory Programs

Welcome to the **Programming in C Language Lab (Practical) \_UHSES0124_AY2025-26** repository! This repository contains C programs designed for students learning the **C programming language**. Each folder corresponds to a specific topic in C, and the programs are organized to help you understand concepts step by step.

---

## Directory Structure and Contents

### 05 To Study Concepts of Decision Making and Looping

This folder contains programs that demonstrate decision-making statements (`if`, `switch`) and loops (`for`, `while`, `do-while`).

| Program                       | Description                                                  |
| ----------------------------- | ------------------------------------------------------------ |
| 01_Palindrome.C               | Checks whether a number entered by the user is a palindrome. |
| 02_Even_or_Odd.C              | Checks whether a number is even or odd.                      |
| 03_Prime_number_in_sequence.C | Lists all prime numbers between 1 and n (entered by user).   |
| 04_Fibonacci_Series.C         | Prints Fibonacci series up to a number entered by the user.  |

---

### 06 To Study User Defined Functions

This folder focuses on creating and using **user-defined functions**.

| Program               | Description                                                                                         |
| --------------------- | --------------------------------------------------------------------------------------------------- |
| 01_Array.C            | Accepts an integer array and prints the array using a function.                                     |
| 02_Basic_Calculator.C | Menu-driven program performing addition, subtraction, multiplication, and division using functions. |
| 03_Lower_to_Upper.C   | Converts a lowercase character to uppercase using a function.                                       |
| 04_Swap_by_value.C    | Swaps two numbers using call by value.                                                              |

---

### 07 To Study Concepts of Arrays

Programs in this folder involve working with arrays.

| Program                    | Description                                                                    |
| -------------------------- | ------------------------------------------------------------------------------ |
| 01_Array_Sum_Avg.C         | Calculates the sum and average of 10 elements of an array.                     |
| 02_Array_Small_Large.C     | Finds the smallest and largest numbers in an array.                            |
| 03_String_Palindrome.C     | Checks whether a given string is a palindrome using string handling functions. |
| 04_Character_Replacement.C | Replaces a specified character in a string with another character.             |

---

### 08 To Study Multidimensional Arrays

This folder contains programs for **2D arrays (matrices)**.

| Program                    | Description                                                         |
| -------------------------- | ------------------------------------------------------------------- |
| 01_Matrix_Operations.C     | Performs addition, subtraction, and multiplication of two matrices. |
| 02_Upper_Triangular.C      | Calculates the sum of upper triangular elements of a matrix.        |
| 03_Matrix_Transpose.C      | Generates the transpose of a matrix.                                |
| 04_Diagonal_Elements_Sum.C | Calculates the sum of primary diagonal elements of a 2x2 matrix.    |

---

### 09 To Study Concept of Pointers

Programs to understand pointers and their usage.

| Program                | Description                                                                |
| ---------------------- | -------------------------------------------------------------------------- |
| 01_Swap_by_reference.C | Swaps two numbers using call by reference.                                 |
| 02_Array_Pointer.C     | Accepts an array using pointer, modifies elements, and displays the array. |

Programs focusing on **dynamic memory allocation**.

| Program                   | Description                                                                                             |
| ------------------------- | ------------------------------------------------------------------------------------------------------- |
